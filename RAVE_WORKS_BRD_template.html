1. Executive Summary

RAVE Works is a social-first creative marketplace connecting Rave Heads (creators, freelancers, editors, musicians, models, sales agents) with OG Vendors (brands, agencies, labels). The platform blends a social feed with a professional work/campaign engine — think LinkedIn × Instagram for the creator economy.

RoleDescription
Rave HeadCreative talent — posts work, applies to campaigns, delivers projects
OG VendorBrand or agency — posts campaigns, reviews applicants, hires creators
AdminPlatform operator — verifies identities, moderates content

7. Key User Journeys

Journey 1 — New Rave Head (Creator)

Sign Up → Choose "Rave Head" → Fill profile + Upload Government ID
       ↓
Dashboard (pending verification banner shown)
       ↓
Admin reviews ID → Approves → verificationStatus = "verified"
       ↓
Creator can now: Post content, Apply to campaigns
       ↓
Browse feed → Find CAMPAIGN post → Click APPLY → Write proposal
       ↓
Vendor reviews application → Clicks HIRE
       ↓
Project workspace created → Creator downloads brief + brand assets
       ↓
Creator submits deliverable URL → Vendor approves → Payment RELEASED

Journey 2 — New OG Vendor (Brand)

Sign Up → Choose "OG Vendor" → Fill brand details + Upload ID
       ↓
Dashboard (pending verification)
       ↓
Admin approves → Campaign posting enabled
       ↓
Create Campaign → Published to global feed
       ↓
Rave Heads apply → Review applicant profiles in Dashboard
       ↓
HIRE creator → Project workspace created
       ↓
Review submission → Approve or request Revision → COMPLETED

Journey 3 — Admin Moderation Cycle

Login as admin → /admin
       ↓
Overview tab: Review platform health metrics
       ↓
Verifications tab: Review pending ID documents → Approve or Reject
       ↓
Users tab: Monitor user activity → Ban bad actors
       ↓
Content tab: Review reported or flagged posts → Delete violations

  ▼                   ▼
